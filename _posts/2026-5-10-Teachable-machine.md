## Un clasificador sencillo con Teachable Machine

Con esta aplicación quise explorar una ruta más simple para poder entrenar modelos de IA para la clasificación de imágenes, esta vez usando Teachable Machine una herramienta online creada por google que es sumamente práctica y accesible, ya que nos evita tener que montar un espacio de trabajo complejo en python con environments, programas, librerias y versiones incompatibles.

El caso de uso es el mismo que el del [clasificador de tipologías fotograficas](https://gustavolsj.github.io/Clasificador-imagenes/), y está basado en el mismo conjunto de datos que recopilé.

## Funcionamiento

Teachable Machine es una pagina web que tiene una interfaz muy sencilla quep permite entrenar un modelo de IA a partir de ejemplos cargados por el usuario y exportarlo para usarlo en tupropía pagina en la web.
