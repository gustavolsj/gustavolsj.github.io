## Clasificación automática de fotografías

Este experimento lo desarrollé para probar cómo un clasificador puede apoyar la identificación de grupos de imágenes a partir de patrones visuales. Aunque no reemplaza la revisión técnica especializada, sí puede acelerar el trabajo preliminar cuando se tienen grandes volúmenes de fotografías.

<!--more-->

## Resultado

La aplicación está disponible aquí [https://gustavolsj.github.io/clasificador_img/](https://gustavolsj.github.io/clasificador_img/)

## Funcionamiento

El sistema utiliza un modelo entrenado con ejemplos previos para reconocer similitudes entre nuevas imágenes y asignarlas a una categoría probable. El resultado es una predicción que sirve como punto de partida para la revisión humana.

En contexto archivístico y de conservación, este enfoque puede ayudar a priorizar revisiones, detectar conjuntos coherentes y reducir tiempos de clasificación manual en etapas iniciales del trabajo.
